# Build App

build local:  
>`docker-compose up -d` 

url local -> [https://localhost:4200](https://localhost:4200)
  

build with k8s:  
>`kubectl apply -f k8s-app.yml.`

NS: metra-bellet-ns

url k8s-> [metra-bellet-ns.k8s.granux.fr](https://metra-bellet-ns.k8s.granux.fr)
